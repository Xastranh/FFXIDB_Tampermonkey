# FFXIDB_Tampermonkey
Tampermonkey Script to fix Google Maps implementation on FFXIDB, this only works for maps that are loaded properly, a lot of maps are indexed incorrectly and won't show or will show things that do not make sense (for example Al Zhabi city shows a lot of mobs).
This was tested in Firefox only, I make no guarantees this will work on your machine / browser.
